# Math Controller Addon
![img3](<images/img3.jpg>)
## 説明
　このアドオンは、記入した数式の通りに数値計算をする拡張mathノード「**Math Controller**」を追加します。  
## Overview
  This addon add **Math Controller**. This is Advanced math node that acts acoording to fromulas your Written.

## Formula list
現バージョンでは、通常のmathノードの関数に対応しています。

# インストール:
現在：blender 5.0.1以上で動作可能です。
リリースからダウンロードして、blender画面にドラッグアンドドロップして下さい。もしくはローカルファイルから読み込んでください。
# Installation:
version：blender 5.0.1 or later
Please downroad this addon from release page, and drag and drop it in your blender, also you can install from local dict.
